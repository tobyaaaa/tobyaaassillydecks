Hello! This is a balatro mod I've had mostly-finished for Actually A Year Now. 
The initial concept was to try and do something similar to Isaac's tainted characters, remixes of the main characters that work a little weirder. I was not able to come up with effects I was happy with for every deck (it turns out ghost deck was remarkably resistant to remixing, for example) but I am VERY happy with what's there.

## Content
11 decks
- Evil red deck: vanilla stakes apply in reverse (red steck gives money on small blind, but has rentals...).
- Evil blue deck: slightly lower base hand size, but each hand you play in a round gives you more for that round.
- Evil yellow deck: start with 0$, but a coupon tag.
- Evil green deck: every time you select a blind, your money is set to exactly 8$.
- Evil magic deck: the first option in arcana packs is always the Emperor.
- Evil nebula deck: arcana packs only contain the Fool.
- Evil abandoned deck: one card of each rank is missing. Slightly random, but will always be very fourth card of each suit.
- Evil checkered deck: start with an *extra* card for each rank. Their suit is random but all are Wild cards.
- Evil zodiac deck: start with overstock, rerolls cost $2 more.
- Evil painted deck: -1 joker slot, and start run with a Trance card (that's the blue seal one).
- Evil anaglyph deck: shop rerolls also reroll the ante's remaining tags.
